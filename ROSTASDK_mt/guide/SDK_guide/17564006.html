Руководство пользователя SDK RC-47 1.0 : Утилиты

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Системную программу, обеспечивающую управление хост-компьютером и платой RC-47 будем называть утилитой. Утилиты решают сервисные задачи и представляют собой готовые к использованию программы, использующие вызовы библиотеки API RC-47. Утилиты реализуются на стороне хост-компьютера с помощью вызовов библиотеки API RC-47, которая взаимодействует с встроенным программным обеспечением (ВПО)  платы RC-47. 

Ниже приведены основные утилиты для работы с платой RC-47.
