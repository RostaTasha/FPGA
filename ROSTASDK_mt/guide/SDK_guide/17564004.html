Руководство пользователя SDK RC-47 1.0 : Программное обеспечение хост-компьютера

Created by Yuri Rumyantsev on Jan 27, 2015

Программное обеспечение хост-компьютера (RC-47 SDK) представляет собой набор библиотек, утилит  и драйвер для работы с платой RC-47.С помощью RC-47 SDK пользователь блока может создавать высокопроизводительные приложения, использующие ПЛИСв качестве аппаратных ускорителей.

Идеология использования ПЛИС для ускорения высокопроизводительных вычислений заключается в совместной и параллельной работе приложения, исполняющегося на хост-компьютере (хост-программа) и аппаратного вычислительного ядра (проект ПЛИС) , реализованного в ПЛИС, которые обмениваются данными между собой по шине PCI Express. При использовании RC-47 SDK от пользователя скрывается вся сложность организации обмена данными по шине PCI Express. Вместо этого ему предлагается простая и отлаженная модель передачи данных, использующая простые документированные библиотечные функции на стороне хост-программы и аппаратные интерфейсы типа регистрового файла, очереди FIFO или AXI шины на стороне ПЛИС. 

Кроме того в состав SDK RC-47  включены несколько полезных сервисных функций -утилит для работы с блоком RB-8V7, таких как температурный мониторинг блока через PCI Express, обновление стартовых конфигураций ПЛИС в карте microSD (установленной на плате RC-47), а также реконфигурация ПЛИС через PCI Express.
