Руководство пользователя SDK RC-47 1.0 : Обнаружение плат RC-47

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Утилита сканирует вычислительный узел по шине PCI Express на наличие в нем плат RC-47 и выводит на экран следующую информацию:

  • количество плат RC-47 в  вычислительном узле
  • для каждой платы RC-47 выводятся:
    • номер платы, присвоенный ей RC-47 SDK
    • список PCI Express устройств на плате RC-47, вместе с их адресами на шине PCI Express и идентификаторами Chip ID 
    • серийный номер
    • версия прошивки микроконтроллера STM 
    • MAC адрес блока RDM

Вызов  в ОС Linux

Запустить утилиту "обнаружение плат RC-47" на хост-компьютере из ОС Linux можно, набрав в командной строке следующее: 

$ cd $ROSTA_SDK_DIR/lin/utilities
$ sudo ./rc47_find_devices

Входные и выходные данные

Утилита не требует входных данных , выходные данные  - информация о подключенных платах RC-47. 

Ниже приведен пример вывода информации о найденной плате.
