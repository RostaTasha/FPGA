Руководство пользователя SDK RC-47 1.0 : Вычислительный узел

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Описание вычислительного узла

Вычислительный узел предназначен для проведения вычислений на основе программируемых логических интегральных схем (ПЛИС) семейства Virtex-7. Вычислительный узел состоит из хост-компьютера с подключенным блоком или блоками RB-8V7 (рис. 1). Для этого в хост-компьютер должна быть установлена плата RHA-25, к двум выходным портам которой посредством двух оптических кабелей подключается блок RB-8V7. Более подробно о подключении блока RB-8V7 см в РПВЕ.466514.001РЭ "Руководство по эксплуатации вычислительного блока RB-8V7". Плата RHA-25 является адаптером расширения PCI Express шины хост-компьютера. 

Возможно подсоединение нескольких блоков RB-8V7 к одному хост-компьютеру.

 

Вычислительный узел с одним блоком RB-8V7

 

Хост-компьютер

Под хост-компьютером понимается компьютер или сервер, с установленным адаптером или адаптерами RHA-25 шины PCI Express РПВЕ.467149.006. Хост-компьютер выполняет следующие функции:

  • служит "root complex" шины PCI Express для подключенных PCI Express устройств в вычислительном узле
  • управляет передачей данных между различными ПЛИС
  • управляет передачей данных между ПЛИС и оперативной памятью хост-компьютера
  • обеспечивает обмен, хранение и обработку данных, поступающих от ПЛИС 
  • выполняет утилиты (системные программы, обеспечивающих управление платами RC-47)

 

Вычислительный блок RB-8V7

Блок RB-8V7 представляет собой автономное устройство с собственным блоком питания и системой воздушного охлаждения. Блок RB-8V7 работает под управлением внешнего хост-компьютера, к которому можно подключать один или несколько блоков RB-8V7. 

Основными вычислительными элементами блока RB-8V7 являются две вычислительные платы RC-47, на каждой из которых установлены по четыре ПЛИС Virtex-7, соединенных между собой шиной PCI Express. Таким образом в составе одного вычислительного узла к хост-компьютеру подключается восемь и более (кратно восьми) ПЛИС Virtex-7.

Вычислительный блок RB-8V7 работает под управлением внешнего хост-компьютера и выполняет следующие функции:

  • проведение вычислений на ПЛИС семейства Virtex-7
  • обеспечивает обмен данными между ПЛИС и хост-компьютером
  • обеспечивает обмен данными между ПЛИС в рамках одного вычислительного узла

Более подробную информацию о составе, подключении и эксплуатации блока RB-8V7 можно найти в "РПВЕ.466514.001РЭ. Руководство по эксплуатации блока RB-8V7.

 

Плата RC-47

Блок RB-8V7 содержит две одинаковые симметрично расположенные платы RC-47 (рис.2). Вычислители ПЛИС установлены непосредственно на плату RC-47, которая представлена основным программным объектом в рамках SDK. Отсюда и название - SDK RC-47. 

 

Плата RC-47

 

На плате RC-47 установлены четыре FPGA Virtex-7 (С0 - С3), коммутатор PCI Express и система жизнеобеспечения LSS (Life Support System). Плата RC-47 имеет два PCI Express соединителя КС1 и КС2. Соединитель КС1 используется для подключения платы RC-47 к хост-компьютеру, а соединитель КС2 для возможного подключения других PCI Express устройств. В качестве PCI Express устройств можно использовать платы RC-47 в блоках RB-8V7 для масштабировании вычислительной системы. Каждая FPGA Virtex-7 имеет свой интерфейс PCI Express x4 Gen2 к коммутатору PCI Express.

Система жизнеобеспечения решает задачи управления питанием платы, контролем температуры и задачи, связанные с конфигурацией ПЛИС Virtex-7. Ее важными элементами являются:

  • Системная ПЛИС Spartan-6, имеющая PCI Express 1.0 gen1 интерфейс к коммутатору PCI Express
  • Микроконтроллер STM32F2
  • ПЗУ для сохранения стартовых конфигураций ПЛИС в виде карты microSD

Система жизнеобеспечения работает под управлением встроенного программного обеспечения (в дальнейшем ВПО платы RC-47), с которым хост-компьютер взаимодействует по шине PCI Express.

Более подробную информацию о составе, устройстве и принципах работы платы RC-47 можно найти в "РПВЕ.466514.001РЭ. Руководство по эксплуатации блока RB-8V7.

 

Программно-аппаратные требования 

Для работы с блоками RB-8V7 хост-компьютер должен удовлетворять следующим программно-аппаратным требованиям:

Аппаратные требования

  • Процессор не ниже Intel Core i5. 
  • Оперативной памяти не менее 4 GB.
  •  Диск не менее 128 GB. 
  • Разъем PCI Express материнской платы -  PCI Express 1.0, 2.1, 3.0 x8. 
  • Плата расширения PCI Express RHA-25

В качестве хост-компьютеров могут выступать как настольные рабочие станции, так и серверные машины, поддерживающие установку адаптеров RHA-25. Возможна и удаленная работа с хост-компьютером, которая обеспечивается стандартными средствами ОС Linux.

Для корректной работы с блоком RB-8V7 необходимо, чтобы работа BIOS на материнский платах по нумерации endpoint устройств шины PCI Express выполнялась в полном соответствии с требованиями стандарта PCI Express gen 3.0. В частности, на предмет такого соответствия были протестированы и признаны годными следующие материнские платы:

  • Asus P6T6 WS Revolution
  • Supermicro X9DRW-7TPF

Программные требования

На хост-компьютере должна быть установлена ОС Linux Ubuntu 12.04 x64 с версией ядра 3.5.

Используется компилятор gcc 4.6. 

Опционально 

На адаптере RHA-25 установлена микросхема PEX8725 (коммутатор шины PCI Express) фирмы PLX, внутри которых расположен контроллер DMA. SDK RC-47 поддерживает передачу данных между памятью хост-компьютера  и ПЛИС  с помощью этого контроллера. Для использования  такой возможности на хост-компьютере должно быть установлено стороннее программное обеспечение фирмы PLX - PlxSdk .  Требования к ОС и инструкцию по установке можно посмотреть на сайте производителя http://www.plxtech.com/sdk

Требования к среде сборки ПЛИС проектов

Проекты ПЛИС могут собираться на любой рабочей станции (не обязательно на вычислительном узле, описанном выше), на котором работает САПР Xilinx Vivado 2014.2. В частности, проекты ПЛИС, включенные в состав SDK RC-47 собирались на машине с ОС Windows 7 x86_64.
