Руководство пользователя SDK RC-47 1.0 : Библиотека API PCI Express

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Библиотека API PCI Express предназначена для работы с PCI Express устройствами и обеспечивает низкоуровневый доступ к функциональности драйвера. Прикладная программа должна использовать вызовы этой библиотеки при работе с устройствами PCI Express фирмы Роста. Обращение напрямую к вызовам драйвера не предусмотрено.

Библиотека API PCI Express предоставляет широкий набор функций: нахождение, открытие/закрытие устройств, обращение к регистрам устройств, передача данных с прямым доступом к памяти (используя контроллер DMA) и пр. Также, в состав библиотеки входят функции, необходимые для перезагрузки конфигураций ПЛИС во время работы хост-компьютера: логическое отключение/включение PCI Express устройств и рескан PCI Express шины.

Библиотека API PCI Express реализована в ОС Windows и Linux с одинаковым интерфейсом в сторону пользовательского приложения для упрощения портирования одних и тех же вычислительных программ между платформами. 

Структуры данных и функции библиотеки API PCI Express приведены в справочном разделе.
