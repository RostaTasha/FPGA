Руководство пользователя SDK RC-47 1.0 : Температурный мониторинг

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Утилита "Температурный мониторинг" предназначена для чтения температуры каждой ПЛИС на плате RC-47. Чтение температуры производится следующим образом: для выбранной платы RC-47, утилита обращается к ВПО платы и считывает из каждой ПЛИС ее температуру.

Вызов  в ОС Linux

Запустить утилиту "Температурный мониторинг" на хост-компьютере из ОС Linux можно, набрав в командной строке следующее: 

# cd $ROSTA_SDK_DIR/lin/utilities/
# ./rc47_monitor –b [board number]

где board number - номер платы RC-47 (счет начинается с нуля).

Входные и выходные данные

Утилита не требует входных данных , выходные данные  - информация о подключенных платах RC-47. 

 

$ROSTA_SDK_DIR/lin/utilities$ sudo ./rc47_monitor -b 0
Searching for RC47 boards...
Found 1 board
Temp PLX = 33
Temp Virtex-7 C0 = 26
Temp Virtex-7 C1 = 24
Temp Virtex-7 C2 = 24
Temp Virtex-7 C3 = 25
