Руководство пользователя SDK RC-47 1.0 : Реконфигурация ПЛИС Virtex-7

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Утилита предназначена для реконфигурации, с хост-компьютера, ПЛИС Virtex-7 через PCI Express интерфейс.  

Утилита сканирует вычислительный узел на наличие в нем плат RC-47 и выбирает одну из них по номеру, указанному пользователем в командной строке. Для выбранной платы утилита обращается к системной ПЛИС и передает ей информацию о том, какие ПЛИС Virtex-7 необходимо реконфигурировать. Утилита логически удаляет PCI Express устройства из системы. После чего передаются конфигурационные данные, считанные из указанного пользователем файла, и через системную ПЛИС осуществляется реконфигурация ПЛИС Virtex-7. После окончания реконфигурации утилита инициирует сканирование шины PCI Express, и ОС находит и настраивает новые PCI Express устройства. 

Вызов  в ОС Linux

Запустить утилиту "Реконфигурация ПЛИС Virtex-7" на хост-компьютере из ОС Linux можно, набрав в командной строке следующее: 

$ cd $ROSTA_SDK_DIR/lin/utilities
$ sudo ./rc47_config -b [board number] [chip_select] -f [path_to_file]

где board number - номер платы в системе, chip_select - любая комбинация C0, C1, C2, C3 или all,  path_to_file  - путь к файлу прошивки *.bit

Входные и выходные данные

Входные данные для утилиты - номер платы RC-47 в системе, кристаллы данной платы, которые необходимо  реконфигурировать, путь к прошивке, которую необходимо загрузить.

Выходные данные  - информация об успешности реконфигурирования выбранных кристаллов, а также  информация о подключенных платах RC-47 после реконфигурирования.
