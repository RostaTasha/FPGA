Руководство пользователя SDK RC-47 1.0 : Библиотека API RC-47

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Библиотека API RC-47 SD специфична именно для работы с платой RC-47 и решает следующие задачи:

  • логическое объединение нескольких PCI Express устройств в структуру rc47_board_t
  • обеспечение взаимодействия между пользовательской программой и ВПО платы RC-47
  • обеспечение взаимодействия между пользовательской программой и DMA контроллером 8725 PLX через вызовы функций из библиотеки  PLX
  • создание файла прошивки для microSD на основе  конфигурационных файлов для системной FPGA и FPGA Virtex7

Также в текущей версии SDK временно библиотека RC-47 API включает в себя некоторые функции для передачи данных между RAM хост-компьютера и внутренней AXI шиной кристалла Virtex-7 (для тех проектов, где она используется). 

Библиотеки API PCI Express и RC-47 API реализованы в ОС Windows и Linux с одинаковым интерфейсом в сторону пользовательского приложения для упрощения портирования одних и тех же вычислительных программ между платформами. 

Структуры данных и функции библиотеки API RC-47 приведены в справочном разделе.
