Руководство пользователя SDK RC-47 1.0 : Общие сведения

Created by Yuri Rumyantsev on Feb 03, 2015

Программные средства, входящие в состав SDK RC-47 предназначены для разработки прикладных вычислительных программ, использующих ПЛИС в качестве аппаратных ускорителей и работающих на вычислительном узле, состоящим из хост- компьютера и блока RB-8V7, подключенного к нему по шине PCI Express. В данной главе

  • Приводятся сведения об аппаратном обеспечении, т.е. вычислительном узле и входящим в него блоке RB-8V7. 
  • Описывается назначение SDK RC-47 и состав его программных компонентов. 
  • Дается информация по особенностям использования шины PCI Express, описываемыми программными средствами.
