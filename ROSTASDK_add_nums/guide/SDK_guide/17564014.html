Руководство пользователя SDK RC-47 1.0 : Драйвер

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Драйвер устройств шины PCI Express работает со всеми изделиями производства НПО "Роста" использующими шину PCI Express. Поддерживаются 64х битные версии драйверов под Windows (XP, 7) и Linux (версия ядра 2.6.38 - 3.5.0). Драйвер ищет устройства PCI Express по уникальному номеру принадлежащему НПО "Роста". Этот номер хранится в регистрах PCI Express устройств  Vendor ID и Device ID:

static struct pci_device_id rsp_board_pci_tbl[] = { 
		{ 0x1819, 0x0527, PCI_ANY_ID, PCI_ANY_ID, 0, 0, 0 }, 	// Virtex-6 на модуле RSP-527
		{ 0x1819, 0x0529, PCI_ANY_ID, PCI_ANY_ID, 0, 0,	0 }, 	// Virtex-6 на модуле RSP-529
		{ 0x1819, 0xB003, PCI_ANY_ID, PCI_ANY_ID, 0, 0, 0 },   	// Системный кристалл Virtex-5 на плате MB-R3
		{ 0x1819, 0x0747, PCI_ANY_ID, PCI_ANY_ID, 0, 0, 0 }, 	// Virtex-7 на плате RC-47
		{ 0x1819, 0xB007, PCI_ANY_ID, PCI_ANY_ID, 0, 0, 0 }, 	// Системный кристалл Spartan-6 на плате RC-47
		{ 0, } };

где первое число является Vendor ID (0х1819), а второе число является Device ID.

Vendor ID - номер разработчика (НПО Роста)

Device ID - номер изделия PCI Express 

Драйвер обеспечивает:

  • отображение BAR регионов в Memory Space
  • доступ к регистрам устройства через BAR
  • Scatter-Gather DMA передачу данных между оперативной памятью RAM хост-компьютера и внутренней логикой устройства
  • поддержку прерываний Legacy Intx

Функциональности Linux и Windows драйверов в текущей версии RC-47 SDK одинаковые.
