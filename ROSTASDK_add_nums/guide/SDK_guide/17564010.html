Руководство пользователя SDK RC-47 1.0 : Сброс конфигурации ПЛИС Virtex-7

Created by Yuri Rumyantsev, last modified by Sergey Torchigin on Apr 27, 2015

Утилита предназначена для сброса конфигурации ПЛИС  по шине PCI Express.

Утилита сканирует вычислительный узел на наличие в нем плат RC-47 и выбирает одну из них, указанную пользователем в командной строке. Для выбранной платы утилита обращается к системной ПЛИС и передает ей информацию о том, какие конфигурации ПЛИС Virtex-7 необходимо сбросить. Утилита логически удаляет PCI Express устройства из системы. Системная ПЛИС сбрасывает конфигурации выбранных ПЛИС Virtex-7. 

Вызов  в ОС Linux

Запустить утилиту "Сброс конфигурации ПЛИС Virtex-7" на хост-компьютере из ОС Linux можно, набрав в командной строке следующее: 

$ cd $ROSTA_SDK_DIR/lin/utilities
$ sudo ./rc47_reset_v7 –b [board number] –v [chip select]

где board number - номер платы в системе, chip_select - любая комбинация C0, C1, C2, C3 или all

Входные и выходные данные

Входные данные для утилиты - номер платы RC-47 в системе, кристаллы данной платы, конфигурацию которых необходимо сбросить.
